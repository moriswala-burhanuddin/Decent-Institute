Advanced Web Technology Course-2 PHP & MySQL

What You'll Learn

Explore advanced backend technologies with PHP and MySQL, enabling you to build dynamic and data-driven websites.

  • Master PHP fundamentals and syntax
  • Handle HTML forms with PHP
  • Implement decisions and loops for logic control
  • Work with functions, strings, and arrays
  • Connect and interact with MySQL databases
  • Build real-world projects like an E-commerce website and online voting system

Course Modules

Module 1: Introduction to PHP

Basics of PHP, evaluation of programs, syntax, and variables.

Module 2: Handling Forms

Capturing form data, dealing with multi-value fields, and file uploads.

Module 3: Decisions & Loops

Conditional statements, loops, and their integration with HTML.

Module 4: Functions

Defining and using functions, call by value/reference, recursive functions.

Module 5: Strings & Arrays

String operations, searching/replacing, and working with arrays.

Module 6: Database Connectivity

Introduction to RDBMS, connecting PHP with MySQL, executing queries.

Mini Projects

  • Develop a US Trip Form
  • Build a Food Order Website using PHP & MySQL
  • Create an E-commerce website
  • Develop an Online Voting System
